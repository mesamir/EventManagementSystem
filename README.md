# Event Management System (EMS)

A modular, database-driven web application designed to streamline event planning, vendor coordination, and guest management in Nepal. Built using Java/JSP, Servlets, MySQL, and Tailwind CSS, EMS supports real-time RSVP tracking, secure Stripe-based payments, and admin-controlled vendor approvals.
